Your first pull request

So, you've discovered Hacktober Fest and thought "I want one of those cool shirts" or maybe you're just here fore the experience of contributing to open source. My guess is it's the latter.

Regardless, you're here because you want to know how to make your first pull request so here's how:

  1. First, you'd want to fork the repository you want to contribute to.
  2. Next, bring the repo to your local machine using git clone.
  3. Then, add your contribution! (make sure it's not just adding your name to a html file as hacktoberfest won't accept this as a contribution, I learnt that the hard way).
  4. stage and commit your changes
  5. push them to the remote
  6. create a pull request and wait for it to be reviewed by someone.

Feel free to add to this file as part of your contribution toward hacktoberfest. Go crazy, add images, make it more in-depth, add links... whatever you want!
